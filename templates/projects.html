Cup Game Program

Created a simple program that allows a user to play a simple cup guessing game with an accompanying gui. The player starts the game and the cups are shuffled on screen. They must they guess which cup a ball has been placed under. The game continues until the user closes the program.

Little Library

For my Eagle Scout project, I built two outdoor libraries and a bench to go outside Portage West Middle School. I pre-stocked the outdoor libraries with books, and the kids can take and leave books as they want. I built it in 2018, and it is still there and in use to this day.
